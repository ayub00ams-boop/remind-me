# Remind Me (মনে রাখো)

Bangla/English reminder app with natural language parsing.

## Features

- NLP Parser for Bangla & English
- Android notifications (AlarmManager)
- SQLite storage
- Dark theme UI

## Build

```bash
flutter pub get
flutter build apk --debug
```

## Usage

- Type: `আমাকে ৫ মিনিট পর মিটিং এ join করার কথা মনে করাইবা`
- Or: `Remind me in 5 minutes`
